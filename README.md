Moustapha Merhi — Portfolio

Personal portfolio for Moustapha Merhi (MX Studios). Single static page, no build step.
The projects section pulls live from GitHub (mymehi21) and updates itself on every push.

Profile picture

Drop any image named profile.jpg into the repo root. The site loads it automatically.
If no profile.jpg is present, it falls back to the GitHub avatar, then to initials.

Deploy (GitHub Pages)

This is a user site, so it serves from the repo root at https://mymehi21.github.io.
Just push index.html (and profile.jpg) to the main branch.
